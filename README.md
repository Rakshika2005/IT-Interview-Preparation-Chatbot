# IT Interview Preparation Chatbot

## 📌 Project Overview

The **IT Interview Preparation Chatbot** is a Python-based console application designed to help students prepare for technical interviews. It simulates a real interview process by conducting multiple rounds and providing feedback based on performance.

This project is useful for students preparing for campus placements and technical interviews.

---

## ✨ Features

- 🔐 User Registration and Login
- 🧠 Aptitude Round with Multiple Choice Questions
- 💻 Coding Round with automatic evaluation
- 📚 DSA (Data Structures & Algorithms) Round
- 🗣️ Group Discussion (GD) Round
- 👨‍💼 HR Interview Round
- 📊 Final Score Dashboard
- 🎯 Performance Feedback
- 🎲 Randomized Questions

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/Rakshika2005/IT-Interview-Preparation-Chatbot.git
```

### 2. Open the project folder

```bash
cd IT-Interview-Preparation-Chatbot
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the project

```bash
python it_interview_chatbot.py
```
